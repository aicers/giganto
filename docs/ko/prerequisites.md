# 설치 전 준비

## 필수 준비 사항

- 서버 인증서, 개인 키, 신뢰할 CA 인증서(PEM 형식)
- `data_dir`는 사전에 필수로 생성
- `export_dir`는 지정된 경로에 디렉터리가 없으면 자동으로 생성

## 네트워크 및 보안 준비

Giganto의 GraphQL 서버는 HTTPS로 동작하며 mTLS 클라이언트 인증을 요구합니다. 따라서 서버를 올리는 것만으로는 충분하지
않고, 클라이언트 측에도 인증서/키/CA 준비가 필요합니다.

## 인증서 준비 원칙

- 서버 인증서와 개인 키는 Giganto 실행 시 필수입니다.
- GraphQL API에 접속하는 클라이언트도 mTLS용 인증서를 준비해야 합니다.
- CA 인증서는 `--ca-certs`로 전달합니다.
- 여러 CA 인증서를 사용할 때는 쉼표로 구분하거나 인자를 반복할 수 있습니다.

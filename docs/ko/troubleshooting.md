# 문제 해결

## 프로세스가 시작되지 않을 때

- `data_dir`가 실제로 존재하는지 확인합니다.
- 인증서, 개인 키, CA 인증서 경로가 올바른지 확인합니다.
- 로그 경로가 지정되었다면 해당 파일에 쓰기 권한이 있는지 확인합니다.

## GraphQL 접속이 되지 않을 때

- 클라이언트 인증서를 함께 제시하고 있는지 확인합니다.
- 서버 실행 시 전달한 `--ca-certs`가 클라이언트 인증서 검증에 사용되므로, 인증서 체인이 일치하는지 확인합니다.

## 클러스터가 정상 연결되지 않을 때

- `peer_srv_addr`와 `peers` 값이 실제 네트워크와 일치하는지 확인합니다.
- peers 간 방화벽 정책과 라우팅을 점검합니다.
- 노드별 인증서 및 이름 정책이 상호 검증 요구 사항과 맞는지 확인합니다.
